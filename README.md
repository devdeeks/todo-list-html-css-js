# To-Do List Project

This is a simple To-Do List web app built with HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Mark tasks as complete
- Delete tasks

## How to Use
1. Download or clone this repository.
2. Open `index.html` in your browser.
